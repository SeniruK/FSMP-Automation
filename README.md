# FSMP Automation Tool

## Overview
The B-1 Force Structural Maintenance Program (FSMP) report is produced annually to define the inspection and maintenance schedule for the B-1 fleet. The ASIP team uses individual aircraft flight data to create the inspection schedule. Previously, the ASIP team processed the flight data with an Excel workbook. This workbook is bloated, riddled with errors, and crashes often.

To address these issues, I created the FSMP Automation Tool. This tool automates the process for calculating the inspection schedule and producing tables for the report.

## Prerequisites for Installation
You will need the following applications/libraries installed to use the FMSP Automation Tool:
* Python 3.8 or later
* Microsoft Access 2013 or later
* Flask
* Pandas

## How to use
To run this tool:
1. Run the main.py function
2. Select the year you wish to analyze
3. Click "Next"
4. Click "Run FSMP Calculations"
